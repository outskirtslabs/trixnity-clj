= ol.trixnity.event

Accessors and predicates for normalized event maps.

This namespace provides convenience accessors over
the normalized event shapes defined in xref:api/ol-trixnity-schemas.adoc[`ol.trixnity.schemas`] without
exposing callers to raw bridge internals.

Use these helpers when consuming timeline events, notifications, or reply
metadata from xref:api/ol-trixnity-room.adoc[`ol.trixnity.room`], xref:api/ol-trixnity-notification.adoc[`ol.trixnity.notification`], and
xref:api/ol-trixnity-room-message.adoc[`ol.trixnity.room.message`].
